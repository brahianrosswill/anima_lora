SPD

Spectral Progressive Diffusion — 学習不要の多解像度推論 (Xiao et al., arXiv:2605.18736)。ノイズ除去の軌跡に沿って空間解像度を拡大します: ノイズが支配的な初期ステップを低解像度で実行し、ノイズから細かい周波数が現れたらスペクトルノイズ拡張で高周波ディテールを注入します。潜在パワースペクトルは冪乗則(P_ω ∝ |ω|^−β、Anima では β ≈ 2.26)で減衰するため、高周波は信号が少なく、後回しにしても安価です。

これは重みアダプタではなくサンプラーレベルの推論モードです — --spd (または make test SPD=1) で有効化します。素の DiT でも既存の LoRA チェックポイントでも動作します。スケジュールは --spd_stages (昇順の解像度スケール、例: 0.5 1.0) と --spd_transition_sigmas (次のステージへ拡張する σ 閾値) で設定します。

2 つの経路。 Case A は完全に学習不要です — ここでは何も学習しません。Case B は任意の軌跡アダプタのファインチューン (make exp-spd) で、モデルが多解像度の軌跡を直接見るように SPD 速度ターゲットで学習した通常の LoRA です。出力はベイク可能な通常の LoRA です。

v0 の制限。 Euler のみ (確率的サンプラーは警告付きでフォールバック)、かつ Spectrum / DCW / SMC-CFG とは組み合わせられません — SPECTRUM=1 SPD=1 はエラーになります (どちらもノイズ除去ループを置き換えるため)。docs/experimental/spd.md を参照。
