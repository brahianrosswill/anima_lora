SPD

Spectral Progressive Diffusion — training-free multi-resolution inference (Xiao et al., arXiv:2605.18736). Grow the spatial resolution along the denoising trajectory: run the early, noise-dominated steps at low resolution, then inject high-frequency detail via spectral noise expansion once finer frequencies emerge from noise. Because the latent power spectrum decays as a power law (P_ω ∝ |ω|^−β, β ≈ 2.26 on Anima), high frequencies carry little signal and are cheap to defer.

This is a sampler-level inference mode, not a weight adapter — enable it with --spd (or make test SPD=1). It composes with the bare DiT or any existing LoRA checkpoint. The schedule is set by --spd_stages (ascending resolution scales, e.g. 0.5 1.0) and --spd_transition_sigmas (the σ thresholds at which to expand to the next stage).

Two paths. Case A is fully training-free — nothing here trains. Case B is an optional trajectory-adapter fine-tune (make exp-spd): a plain LoRA trained on the SPD velocity targets so the model sees the multi-resolution trajectory. Its output is a normal bakeable LoRA.

v0 limitations. Euler only (stochastic samplers fall back with a warning), and it does not compose with Spectrum / DCW / SMC-CFG — SPECTRUM=1 SPD=1 raises (both replace the denoise loop). See docs/experimental/spd.md.
