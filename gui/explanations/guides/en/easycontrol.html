EasyControl

Extended self-attention image conditioning. The DiT is frozen; training updates only a per-block cond LoRA on self-attn (q/k/v/o) + FFN, plus a per-block scalar gate b_cond (init -10) that controls how much of the reference image's tokens leak into each step's self-attention.

⚠ Phase-1 baseline — not doing anything meaningful yet. This is the ref == target setup, where the condition is the target image. With no real control task to learn, current checkpoints don't produce a useful conditioning effect — treat this variant as plumbing / scaffolding validation, not a usable control adapter. Pick Colorize for an actual control task.

Source data goes in easycontrol-dataset/; caches land in post_image_dataset/easycontrol/. See docs/experimental/easycontrol.md.
